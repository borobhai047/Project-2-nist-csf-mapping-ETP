# Project 2 — NIST Cybersecurity Framework Mapping & Maturity Assessment

## Overview
This project implements the NIST Cybersecurity Framework (CSF) for an enterprise network environment and evaluates cybersecurity maturity across all five CSF functions.

---

## Objectives
- Map existing security controls to NIST CSF
- Assess cybersecurity maturity level (0–4 scale)
- Identify gaps in security posture
- Define improvement roadmap

---

## Environment
**Organization:** Endpoint Technologies PLC  
**Domain:** Enterprise Network Security  
**Scope:** VLAN-based corporate network (IT, HR, DEV, DMZ, GUEST)

---

## Security Frameworks Used
- ISO/IEC 27001
- NIST Cybersecurity Framework (CSF)

---

## NIST CSF Coverage

### Identify
- Asset inventory
- Risk assessment integration

### Protect
- VLAN segmentation
- ACL implementation
- SSH secure access

### Detect
- Port security monitoring
- Network baseline definition

### Respond
- ACL-based containment strategy
- Incident response planning (partial)

### Recover
- Business continuity planning (planned)
- Recovery procedures (in development)
